# CokoNet Academy : Data Analytics Portfolio

- This repository is a collection of my learning  journey at **<a href ="https://www.cokonet.com/">CokoNet</a> Academy**.
- It contains practice scripts, core assignments, and data visualizations developed using Python, R and PowerBI,Tableau.

# Repository Structure

- The projects are organized by the primary tool used :

### Python (Basic Scripting & Data Cleaning)

- Basic syntax, loops, conditional statement and data structures.
- Data Manipulation and Visualization using Pandas, Matplotlib and Seaborn.
- A project on world wide video game sales.

### R Language (Statistical Analysis)

-  Statistical test and data modeling.
- Exploratory data analysis and visuals using 'ggplot'.

### SQL(Database Management & Querying)

- Basic DDL/DML practice (Joins, Aggregations, Subquires).
- Database schema for a school management system.
